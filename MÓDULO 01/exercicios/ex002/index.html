Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer maneira
Basta colocar tudo entre o par de tags <p> e </p>

Se você precisar quebrar o texto em um ponto específico
como agora, você pode usar a tag <br>.

Lista de símbolos

O comando "&+nbsp;", significa espaço em branco;
O comando "&+reg;", significa marca registrada: ®;
O comando "&+trade;", significa trade mark: ™;
O comando "&+euro;", significa moeda Euro: €;
O comando "&+pound;", significa Libra(pound): £;
O comando "&+yen;", significa moeda Yen: ¥;
O comando "&+cent;", significa Centavos: ¢;
O comando "&+empty;", significa vazio: ∅;
O comando "&+sum;", significa somatório: ∑;
O comando "&+Delta;", significa Delta: Δ;
O comando "&+larr;", significa Seta esquerda: ←;
O comando "&+uarr;", significa Seta acima: ↑;
O comando "&+rarr;", significa Seta direita: →;
O comando "&+darr;", significa Seta baixo: ↓;

Lista de emojis

Comando: "&#x+1F596", Resultado:🖖
Comando: "&#x+26F9;", Resultado:⛹
Comando: "&#x+1F3B8;", Resultado:🎸
Comando: "&#x+1F94B;", Resultado:🥋
Comando: "&#x+1F3C0;", Resultado:🏀

Caso queira a lista de todos os emojis disponíveis, o link é: https://emojipedia.org
O site possui atualizações que entram novos emojis, caso não tenha o que procura acompanhe sazonalmente.

Considerações finais

Obs.: O sinal de "+" não significa que deve ser colocado, é apenas para representar que devem ser escritos juntos, sendo o que tem antes do "+" juntamente com o que tá depois do mesmo sinal. Assim como as aspas ("") significam que deve ser escrito apenas o que está dentro delas.
